### 🏢 [Staff Hub — Employee Management System](https://github.com/yazanm16/staff-hup)
> **Laravel · PHP · MySQL · Sanctum · Spatie**

A real-world employee management system built during my internship at **Rentoor (Tahwool)**.

- Architected RESTful APIs using **Service Layer** and **Repository Pattern**
- Enforced **RBAC** with Spatie Permission and **Laravel Sanctum** for token-based auth
- Implemented **background job processing** via Laravel Queues and Job Batching
- Automated weekly attendance reports with **Laravel Scheduler**
- Built a **real-time notification system** using broadcasting
